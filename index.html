SapplyValues

Was ist SapplyValues?

SapplyValues ist ein politischer Kompass Test, der Fragen von Sapply test* mit einer UI von 8values verbindet, hierbei handelt es sich um die deutsche Übersetzung der Seite SapplyValues.github.io**. Dir werden Aussagen gezeigt und du musst mit deiner Meinung dazu antworten, diese reichen von Starke Zustimmung bis Starke Ablehnung, jede Antwort beeinflusst deine Punkte leicht. Am Ende vom Quiz, werden deine Antworten auf einem politisches Kompass angezeigt.

* Die Fragen wurden teilweise umgeschrieben.

** Die Fragen wurden von mir aus dem Englischen übersetzt, dabei kann es zu leichten Bedeutungsänderunge gekommen sein. Es gibt Fragen im Test.

Warum diese Version verwenden?

Sowohl die orginale als auch der beliebte Klon haben einige Probleme und sind sehr schlecht programmiert. Zunächst tritt bei vielen Nutzern ein NaN ("Not a number") Fehler auf. Das passiert wenn man die Tastatur verwendet um Enter zu drücken sund auf die nächste Seite zu kommen anstatt darauf zu klicken. Bei dieser Version passiert das nicht.

Des weiteren geniert die orginale Version kein richtiges Bild. Sie platziert nur ein Overlay über ein CSS-definiertes Hintergrundbild, Was bedeutet, dass wenn du das Bild speichern möchtest, nur das Overlay gespeichert wird und nicht der komplette Kompass. Auch das tritt in dieser Version nicht auf.

Drittens unterstützt diese Version URL-Parameter, das bedeutet du kannst einfach den Link zu deinem Ergebnis teilen und wenn du das Bild verlierst findest du den Link mit Parametern in deinem Browserverlauf, dadurch musst du nicht den ganzen Test wiederholen.

Zuletzt ist die UI von 8values / 9Axes sehr viel besser (zumindest meiner Meinung nach). Sie sieht besser aus und benötigt weniger Klicks um zu antworten.

Du bist unzufrieden mit Fragen oder Antworten!

Wende dich an den Entwickler des orginalen Sapply. Ich habe diesen Test nicht entwickelt nur die UI gemacht.

Datenschutz Anmerkung

Anders als die Englische Version dieser Seite verwenden wir kein Google Analytics und erheben keine der Daten. Politische Meinung ist ein sehr sensibles Thema weshalb der Schutz politischer Daten besonders wichtig ist. Wenn sie sicher gehen wollen können sie auch den Code von Github herrunterladen, überprüfen und lokal auf ihrem Rechner ausführen. Um maximale Transparenz zu gewährleisten muss ich an dieser Stelle anmerken, dass die Seite auf Github gehostet wird, welches zu Microsoft gehört, dadurch sind grundsätzliche Datenschutzhinweise von Github zu beachten.
